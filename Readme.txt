User Guide  
1. Make sure your system have a camera and python installed(version 3.6.5 used).
2. Run createDS.py to create a Data set.
3. Run Trainer.py to train a network.
4. Put your Name  and ID in loop as Facerecog.py.
5. Run Facerecog.py to see AI recogising you.


To Chk basic face detect run facedetecthaar.py.
